Lab: Pixel Art Maker

Design Canvas

Right-click on the image and use Save Image option

Choose Grid Size


Pick A Color

Instructions:

Enter the Grid size using the Grid Height

and Grid Width option. Hit Submit.

Select a color using the color palette

Click and drag inside the grid to draw

Double click to erase single cell OR

Hold CTRL key and drag mouse to erase

multiple cells at once

Clear Grid button will erase entire grid

Hit Save to save your masterpiece

Hit Reset or Refresh page to start anew
